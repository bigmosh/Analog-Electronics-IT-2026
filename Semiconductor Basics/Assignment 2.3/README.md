# 1N4148 Small-Signal Diode

**Diode:** 1N4148  
**Manufacturer:** Vishay Semiconductors  

## Application
The 1N4148 is commonly used in **high-speed signal switching**, such as logic circuits, signal clipping, and protection of sensitive inputs in analog and digital electronics.

This repository documents key electrical characteristics of the diode and explains how they relate to real-world use.
